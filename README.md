

# LornJung Project

This is final year 1 project in CPE121



## Installation

1.Install Node.js

2.Git Clone or Download my project
```bash
  gh repo clone zunzureal/LornJung
  cd LornJung
```
3.Install Package
```bash
  npm install
```
4.Start Metro
```bash
  expo start -c
```
